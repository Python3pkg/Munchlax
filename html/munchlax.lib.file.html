munchlax.lib.file module¶

class munchlax.lib.file.File(slack, file)¶

Bases: munchlax.lib.object.Object

Represents a Slack file.

add_reaction(name)¶

Adds a reaction to the file.

Parameters:name (str) – The name of the reaction to add. A list of standard Slack reactions/emoji can be found at https://www.webpagefx.com/tools/emoji-cheat-sheet/
Raises:SlackError – Raised in the event that Slack does not return ok.
comment(comment)¶

Adds a comment to the file.

Parameters:comment (str) – The comment string.
Returns:A Comment object representing the new comment.
Return type:Comment
Raises:SlackError – Raised in the event that Slack does not return ok.
delete()¶

Deletes the file.

Raises:SlackError – Raised in the event that Slack does not return ok.
get_channels()¶

Fetches and returns all channels this file has been shared in.

This method is a generator.

Yields:Channel – A channel within which the file has been shared.
Raises:SlackError – Raised in the event that Slack does not return ok.
get_comments(count=100, page=1)¶

Fetches and returns the comments for the file.

Parameters:
  • count (int) – The number of comments per page to return. Defaults to 100.
  • page (int) – The comments page number to use. Defaults to 1
Returns:

A list of Comment objects for the query. object: A generic object containing paging information.
Return type:

list (Comment)
Raises:

SlackError – Raised in the event that Slack does not return ok.
get_reactions()¶

Fetches and returns a list of all reactions for the file.

Returns:A list of generic objects with reaction data.
Return type:list (Object)
Raises:SlackError – Raised in the event that Slack does not return ok.
remove_reaction(name)¶

Removes a reaction from the file.

Parameters:name (str) – The name of the reaction to remove.
Raises:SlackError – Raised in the event that Slack does not return ok.
revoke()¶

Revokes the file from being shared publicly.

This will cause the current File object to become stale.

Returns:An updated File object without the public URL for the file.
Return type:File
Raises:SlackError – Raised in the event that Slack does not return ok.
share()¶

Creates a public URL for the file.

This will cause the current File object to become stale.

Returns:An updated File object with the public URL for the file.
Return type:File
Raises:SlackError – Raised in the event that Slack does not return ok.
update()¶

Updates the current File object.

There isn’t much benefit to using this and it’s here if you don’t want to replace your current File object or can’t.

Raises:SlackError – Raised in the event that Slack does not return ok.
