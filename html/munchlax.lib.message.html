reply(text, **kwargs)¶

Writes a reply to the message.

This is basically a convenience function for:

slack.raw_write(text=text, channel=message.channel)

Parameters:
  • text (str) – The text of the message.
  • **kwargs – Additional options to use when sending the message. Refer to Slack#raw_write for more information. In most cases, you will only need to specify text if you only want to send a text message.
Returns:

A Message object representing the newly sent message.
Return type:

Message
Raises:

SlackError – Raised in the event that Slack does not return ok.
