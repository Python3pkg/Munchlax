munchlax.lib.async module¶

munchlax.lib.async.async_wrapper(loop, *args, **kwargs)¶

Executes a function asynchronously.

Parameters:
  • *args – A variadic list of arguments. The first argument should be the function to execute. All other arguments should be arguments for the function.
  • **kwargs – Keyed arguments to pass to the function when executing.
